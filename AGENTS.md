# AGENTS.md(skillink)

你的职责是管理 Skills，以适配不同的 AI 工具。

## 要求

- 在合适地方添加注释，注释使用中文
- 使用中文回复
- 目前代码是 Node 20+，请使用现代化语法
- 技术栈为 TypeScript 开发
- 使用 pnpm 作为包管理工具

## Code style

- TypeScript strict mode
- 使用单引号，有分号

## Setup commands

- 安装使用: `pnpm install`
- build 代码: `pnpm build`
- TS 校验: `pnpm lint`
- Eslint 校验代码: `pnpm check`
- 格式化代码: `pnpm format`
